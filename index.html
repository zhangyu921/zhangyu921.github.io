问答

ajax 是什么？有什么作用？

AJAX = Asynchronous JavaScript and XML, 可以异步的在后台与服务器交换数据，可以在不重载整个网页，对网页进行更新。

前后端开发联调需要注意哪些事情？后端接口完成前如何 mock 数据？(npm install -g server-mock)

需要注意

  • 约定数据传输接口
  • 约定接口参数
  • 约定数据格式

mock数据

  • 可以使用server-mock进行
  • 根据约定的格式进行测试

点击按钮，使用 ajax 获取数据，如何在数据到来之前防止重复点击?

可以设定变量作为标记，当事件被执行的时候，将这个标记置位，执行结束后将标记复位。当事件被重复执行，在事件响应程序中加入判断标记是否被置位，如被置位，直接返回。

问答：

dom对象的innerText和innerHTML有什么区别？

.innerText 和 .innerHTML都是DOM元素的可写属性。

读取innerText的时候，会将该元素包含的的所有文本内容作为字符串返回，若该元素具有下级元素，则会依次进行字符串的拼接。

而.innerHTML是返回HTML结构

通过对.innerText 和 .innerHTML进行赋值，可以改变当前页面元素。
