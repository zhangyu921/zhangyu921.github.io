问答

说说库和框架的区别？

  • 库(Library)，简称Lib，是可以重复利用的代码封装的集合。

    • 开发者可以使用较为简单的操作，通过库内部的代码，实现复杂的效果。
    • 面向对象的代码组织形式而成的库也叫类库。面向过程的代码组织形式而成的库也叫函数库。在函数库中的可直接使用的函数叫库函数。
    • 开发者使用库提供的功能，用来实现新的功能。

  • 框架(Framework)，是用于解决一类问题的完整的产品。

    • 框架用户一般只需要使用框架提供的类或函数，即可实现全部功能。
    • 使用者只需挑选使用框架所提供的功能，解决问题即可。

jquery 能做什么？

jQuery是目前使用最广泛的JavaScript函数库。是对原生JavaScript的拓展，它的功能包括但不限于以下几个方面：

  • 简单快捷的元素选择方法
  • 一系列操作DOM的方法
  • 修改属性及CSS
  • 事件的处理
  • 动画效果
  • Ajax
  • jsonp
    这些通过原生JS都可以实现，但jQuery封装了复杂冗余可以复用的操作，使代码量减少，提升开发效率。

jquery 对象和 DOM 原生对象有什么区别？如何转化？

jQuery对象与DOM对象：

  • DOM对象是通过原生JS提供的方法(例如：document.querySelector(‘body’))获得的对象，而jquery对象是用jquery选择器获得的对象(例如：$(‘body’))
  • jQuery对象是jQuery包装DOM对象后产生的，它可以使用jQuery的方法，无法使用任何DOM对象的方法。
  • 同样DOM对象也不可使用jQuery对象的方法。

相互转化：

  • jQuery提供两了两种方法将jQuery对象转化为DOM对象，即[index]和.get(index)。 jQuery对象本身就是数组对象，使用下标可以直接返回DOM对象。
  • 对于DOM对象，只需要用$()把DOM对象包装起来就可以了。

jquery中如何绑定事件？bind、unbind、delegate、live、on、off都有什么作用？推荐使用哪种？使用on绑定事件使用事件代理的写法？

jQuery绑定事件的方法有简便方法喝通用方法之分。

简便方法是通过jQuery对象的一系列方法，允许直接为常见事件绑定回调函数。

$('.btn').click(function(){
   console.log('click!') 
});

这样的简便方法有：
click
keydown
keypress
keyup
mouseover
mouseout
mouseenter
mouseleave
scroll
focus
blur
resize
hover

通用方法包括 on方法、trigger方法、off方法

on方法

on方法是jQuery事件绑定的统一接口，简便方法是on方法的简写形势。接受两个参数，第一个是事件名称，第二个是回调函数。

$('li').on('click', function (e){
    console.log($(this).text());
});

值得注意的是，在回调函数中的this关键字指的是发生事件的DOM对象。为了使用jQuery方法，必须将DOM转化为jQuery对象。

on方法还可以为当前元素的某一个子元素，添加回调函数。

$('ul').on('click', 'li', function(e){
    console.log(this);
});

这样可以为ul下的li绑定click事件。click事件还是在ul元素上触发，只是在触发的时候会检查event对象的target属性是否为li，如果是li再执行回调函数。这样写，可以对绑定事件之后生成的li依然有效。

trigger方法

trigger方法用于触发回调函数，trigger的中文意思就是扳机。

$('li').trigger('click');

上面代码是执行所有li元素上绑定的click事件回调函数，而且不会触发浏览器默认行为。

off方法

off方法可以移除所有该事件的回调函数。

$('li').off('click');

上面代码移除所有li元素上绑定的click事件回调函数。

bind、unbind、delegate、live方法是在jQuery 1.X版本中使用的方法

  • bind() 向匹配元素附加一个或更多事件处理器
    unbind() 从匹配元素移除一个被添加的事件处理器
    delegate() 向匹配元素的当前或未来的子元素附加一个或多个事件处理器
    live() 为当前或未来的匹配元素添加一个或多个事件处理器

  • 在新版本中以被代替。

jquery 如何展示/隐藏元素？

使用jQuery对象的.hide(),.show()方法。

$('.ct').hide();
$('.ct').show();

可以给hide、show添加参数使之具有动画效果。

jquery 动画如何使用？

jQuery提供了一些动画效果的方法

> show：显示当前元素。
hide：隐藏当前元素。
toggle：显示或隐藏当前元素。
fadeIn：将当前元素的不透明度（opacity）逐步提升到100%。
fadeOut：将当前元素的不透明度逐步降为0%。
fadeToggle：以逐渐透明或逐渐不透明的方式，折叠显示当前元素。
slideDown：以从上向下滑入的方式显示当前元素。
slideUp：以从下向上滑出的方式隐藏当前元素。
slideToggle：以垂直滑入或滑出的方式，折叠显示当前元素。

可以接受毫秒数，或预定义的关键字作为参数。

$('li').show();
$('li').show(500);
$('li').show('slow');//normal, fast 
//注意预定义关键字也是字符串
jQuery.fx.speeds.blazing = 30;//自定义关键字

该方法可以接受第二个参数，作为动画结束的回调函数：

$('li').shou(500, function(){
    alert('done!')
});

jQuery还具有通用的动画效果方法animate

animate可以接受两个参数，第一个参数是一个对象，表示动画结束时相关CSS属性的值，第二个三处是动画持续时间毫秒数，第三个是动画结束的回调函数（可无）。

$('div').animate({
    left: '+=50', // 增加50
    opacity: 0.25,
    fontSize: '12px'
},
300, // 持续时间
function() { // 回调函数
    console.log('done!');
}
);

stop方法，delay方法

  • stop方法立即停止当前元素执行的动画。

  • delay方法可以使动画暂停一段时间后继续执行。

上面代码表示，slideUp动画之后，暂停800毫秒，然后继续执行fadeIn动画。

如何设置和获取元素内部 HTML内容？如何设置和获取元素内部文本？

设置和获取元素内部的HTML元素，可以通过jQuery对象的.html()方法，相当于DOM元素的innerHTML方法。

$('li').html('<img src="img.jpg">');

设置和获取元素内部文本，可以通过jQuery对象的.text()，相当于DOM元素的innerText方法。

$('li').text('Hello, world!');

如何设置和获取表单用户输入或者选择的内容？如何设置和获取元素属性？

使用jQuery元素的val方法，可以设置和获取表单用户输入的内容。

$('input').val();//获取表单用户输入

使用jQuery元素的attr方法，可以设置和获取元素的属性。

$('img').attr('src', '/img/side.jpg');

第一个参数为要设定的属性名称，第二个参数为属性的内容。

removeAttr方法，可以去除元素的属性。

$('img').attr('src');
  • 图片懒加载：在图片元素中设置自定义属性，写进图片地址。当文档浏览到需要显示图片的时候，再把图片地址写进src属性中。
